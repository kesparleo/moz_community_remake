# Bem-vindo(a)

Este repositório reúne e apresenta comunidades de tecnologia e programação em Moçambique.  
O objectivo é centralizar informação, facilitar o contacto entre comunidades e promover o ecossistema tecnológico local.

## Contribuições

Contribuições são bem-vindas e encorajadas.

Regras essenciais:
- O padrão de contribuição adoptado é: `Git Flow`
- Crie sempre uma *feature branch*
- Faça push para `develop`
- Nunca faça push directo para `master`

Pull requests que não respeitem este fluxo não serão aceites.

## Nota

Este projecto está em evolução contínua.  
Sugestões, correcções e melhorias são valorizadas, desde que fundamentadas.

Obrigado pelo interesse e pela colaboração.
